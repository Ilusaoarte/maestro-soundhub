# Maestro SoundHub

Biblioteca de sons CC0 (domínio público) para o app Maestro.

Todos os sons foram obtidos do [Freesound.org](https://freesound.org) com licença **Creative Commons Zero (CC0)** — sem restrições de uso, sem necessidade de atribuição.

## Categorias

| Pasta | Categoria no app | Descrição |
|---|---|---|
| `ambient/` | Ambiente | Atmosferas, loops de fundo, texturas |
| `impact/` | Impacto | Hits, booms, impactos dramáticos |
| `mistery/` | Mentalismo | Suspense, mistério, tensão |
| `stingers/` | Musical | Fanfarras, ta-dás, stingers orquestrais |
| `woosh/` | Transição | Whooshes, risers, swishes |

## Catálogo

O arquivo `sound_catalog.json` é consumido diretamente pelo app Maestro. Não editar manualmente sem atualizar também o `version` e o `updatedAt`.
